# StayNest Hotel Booking

A modern, responsive hotel booking landing page built with plain HTML, CSS, and JavaScript.

## Run locally

Open `index.html` directly in a browser, or serve the folder with a static server such as:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Features

- Hero section with a premium booking feel
- Search form for destination, dates, guests, and budget
- Featured hotel cards with live filtering
- Sticky booking summary with price calculation
- Responsive layout for desktop and mobile
